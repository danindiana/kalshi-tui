# Session Notes — 2026-04-30 00:02:03

## Features Added This Session

### 1. Real-time market-close countdown (`src/ui.rs`)

**What changed:** The Markets tab dashboard previously showed `T-{mins}m to 5 PM EDT` sourced from `model.mins_to_expiry`, a Python subprocess output that only updated every ~30 seconds. Replaced with a live calculation via `compute_secs_to_market_close()` that runs on every 50ms frame draw.

**Format:** `T- HH:MM:SS to close` (or hours when > 1h out). Colors: cyan (> 30m), yellow (10–30m), red (< 10m), gray (MARKET CLOSED).

**Why 21:00 UTC is hardcoded:** 5 PM EDT = UTC-4 → 21:00 UTC. Consistent with the existing "5 PM EDT" label in the codebase. EDT is in effect April–November; if winter hours matter, change the constant to 22 for EST.

**Same countdown also displayed** in the P&L Chart tab summary header so close-time is visible from any tab.

### 2. P&L Chart tab (`src/app.rs` + `src/ui.rs`)

**What changed:** Added `Tab::PnlChart` ("P&L Chart") as the 7th tab. Accessible via Tab/Shift+Tab cycling.

**Layout:**
- 4-line summary header: mode (LIVE/DRY/MIXED), total/settled counts, win rate, open count, net P&L, deployed capital, and market-close countdown
- Full-height `ratatui::widgets::Chart` (braille-marker line graph) below

**Datasets on the chart:**
- Cyan — All settled trades (combined)
- Green — Live trades only
- Yellow — Dry-run trades only

Lines that have no data (e.g., zero live trades ever placed) are omitted to keep the legend clean.

**X axis:** Trade sequence index (sequential 0–N). Wall-clock time was not used because `placed_at` is a raw string in the DB and parsing it for axis labeling would add fragility; trade index is unambiguous.

**Y axis:** Cumulative P&L in dollars. Bounds auto-scale around min/max with 0.5 padding; zero line is the mid-label.

**Open trades excluded:** Trades with `pnl = NULL` (unresolved) do not contribute to the chart lines.

**Data ordering:** `app.auto_trades` is stored DESC (newest first, `ORDER BY t.id DESC`). The chart iterates `.rev()` for chronological accumulation.

**Empty state:** If no settled trades exist, a placeholder message is shown instead of an empty chart.
